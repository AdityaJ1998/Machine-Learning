# Iris Plants Dataset #

* Number of Instances : 150 (50 in each of three classes)
* Number of Attributes : 4 numeric, predictive attributes and the class
* Attribute Information :
 	* sepal length in cm
  * sepal width in cm
  * petal length in cm
  * petal width in cm
* class Information :
  * Iris-Setosa
  * Iris-Versicolour
  * Iris-Virginica
